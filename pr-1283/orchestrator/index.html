Red Hat Developer Hub 1.7

Installing Red Hat Developer Hub with the Orchestrator plugin

Install, configure, and manage Red Hat Developer Hub with the Orchestrator plugin using Helm or the Red Hat Developer Hub Operator

Red Hat Customer Content Services
Legal Notice

Abstract

As a cluster administrator, you can install Red Hat Developer Hub (RHDH) with the Orchestrator plugin.
  • 1. Installing Red Hat Developer Hub with the Orchestrator plugin using Helm
    • 1.1. Installing Red Hat Developer Hub on OpenShift Container Platform in a partially disconnected environment with Helm chart
    • 1.2. Installing Red Hat Developer Hub in a a fully disconnected environment with Helm chart

You can install Red Hat Developer Hub (RHDH) using one of the following methods:

  • The Red Hat Developer Hub Operator
  • The Red Hat Developer Hub Helm chart

1. Installing Red Hat Developer Hub with the Orchestrator plugin using Helm

You can install Red Hat Developer Hub in a fully disconnected or partially disconnected environment using the Red Hat Developer Hub Helm chart.

1.1. Installing Red Hat Developer Hub on OpenShift Container Platform in a partially disconnected environment with Helm chart

You can install Red Hat Developer Hub (RHDH) in a partially disconnected environment by using the Helm chart repository.

Prerequisites

  • You must have access to a cluster running OpenShift Container Platform 4.14 or later, and you must have Helm 3.10 or later installed.

Procedure

  1. Add the chart repositories when you have connectivity as shown in the following example: 

    helm repo add bitnami https://charts.bitnami.com/bitnami
helm repo add backstage https://backstage.github.io/charts
helm repo add redhat-developer https://redhat-developer.github.io/rhdh-chart

  2. Install the chart as shown in the following example:: 

    helm upgrade -i <release_name> redhat-developer/backstage

1.2. Installing Red Hat Developer Hub in a a fully disconnected environment with Helm chart

You can install Red Hat Developer Hub (RHDH) in a fully disconnected environment by using the chart in OCI format from your internal registry.

Prerequisites

You must have access to a mirrored OCI registry that contains the Red Hat Developer Hub chart, and you must have Helm 3.10 or later installed.

Procedure

  1. Install the chart from the OCI registry: 

    helm upgrade -i <release_name> oci://<internal-registry>/rhdh/chart --version=<version>
