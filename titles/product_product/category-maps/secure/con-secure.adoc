= Secure

[role="_abstract"]
Manage authentication and authorization in {product} to control user access, verify identities, and enforce role-based policies.

You can enable authentication in {product} to allow users to sign in using credentials from an external identity provider, such as {rhbk}, GitHub, or {azure-brand-name}, and provision user and group data to the software catalog.

{product} ({product-very-short}) administrators can use role-based access control (RBAC) to manage authorizations of other users by defining roles, permissions, and policies for users and groups.
