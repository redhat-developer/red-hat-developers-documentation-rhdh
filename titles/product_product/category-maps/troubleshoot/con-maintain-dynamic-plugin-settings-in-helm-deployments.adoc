= Maintain dynamic plugin settings in Helm deployments

[role="_abstract"]
Troubleshoot and maintain dynamic plugin configurations in your Red Hat Developer Hub Helm deployments to ensure proper plugin functionality and cache management.

When you deploy Red Hat Developer Hub using Helm charts, dynamic plugin configurations may require ongoing maintenance to address issues with plugin installations, cache settings, and air-gapped environments. These procedures help you manage plugin settings effectively and resolve common deployment-related problems.
