= Inspect plugin logs

[role="_abstract"]
View the {openshift-ai-connector-name} plugins in the `backstage-backend` container to diagnose issues.

.Procedure
. Check the plugin logs for the following components:
+
[cols="3,4,4"]
|===
|Plugin Component |Logger Service Target |Common Log Text

|Model Catalog Entity Provider
|`ModelCatalogResourceEntityProvider`
|`Discovering ResourceEntities from Model Server...`

|Model Catalog TechDoc URL Reader
|`ModelCatalogBridgeTechdocUrlReader`
|`ModelCatalogBridgeTechdocUrlReader.readUrl`
|===

. Optional: To enable debug logging, set the `LOG_LEVEL` environment variable to `debug` on the `backstage-backend` container.
