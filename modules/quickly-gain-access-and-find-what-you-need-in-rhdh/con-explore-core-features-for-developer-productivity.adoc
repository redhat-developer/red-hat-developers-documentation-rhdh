= Explore {product-very-short} core features for developer productivity

As a developer, you can use {product-very-short} as a suite of integrated tools designed to streamline the software development lifecycle. Understanding these core capabilities helps you utilize the platform effectively for self-service and discovery. 

.Developer capabilities

Software Catalog::
A centralized inventory of all software components, APIs, systems, and resources. It allows you to visualize ownership, dependencies, and lifecycle status across the organization.

Software Templates::
A self-service tool that automates the creation of new projects. Templates bootstrap repositories, configure CI/CD pipelines, and apply best practices automatically, eliminating manual setup.

TechDocs::
A "docs-as-code" solution that renders technical documentation directly within the portal. It keeps documentation close to the code, making it easier to find, read, and maintain.

Plugins::
An extensible architecture that connects RHDH to your existing ecosystem. You can view Jira tickets, check ArgoCD deployments, or monitor Quay registry scans without leaving the interface.

Global Search::
A unified search engine that indexes your entire ecosystem, allowing you to find specific components, documentation, or users instantly.
