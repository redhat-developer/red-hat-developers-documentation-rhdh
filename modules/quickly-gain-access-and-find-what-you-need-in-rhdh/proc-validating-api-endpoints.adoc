= Validating API endpoints in {product-very-short}

To verify API behavior and debug responses, use the interactive test interface in {product-very-short}. This ensures endpoints return the correct status codes and data without requiring external testing tools.

.Prerequisites

* You have the necessary credentials if the API requires authentication.

.Procedure

. Navigate to the API you want to test and click the *Definition* tab.
. Locate the operation you want to verify and click to expand it.
. Click *Try it out* to enable the interactive test interface.
. Enter the required parameters or request body JSON in the provided fields.
. Click *Execute*.
. Review the *Server response* section to verify the status code, response body, and headers.

[NOTE]
====
Some APIs require authentication. If you receive 401 or 403 errors, make sure you have valid credentials or contact the API owner.
====
