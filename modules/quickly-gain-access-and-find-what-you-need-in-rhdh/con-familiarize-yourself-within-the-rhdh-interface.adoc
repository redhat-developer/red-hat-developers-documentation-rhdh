= Familiarize yourself within the Red Hat Developer Hub interface

Once logged in, the interface is divided into two primary navigation areas that help you quickly locate resources, tools, and settings.

.Global Header
The top bar provides immediate access to high-frequency actions and personal settings.

Search::
Use the input field to instantly find components, APIs, docs, and users across the entire ecosystem.

Self-Service (+)::
Click the plus icon to quickly access templates for creating new components or projects.

User Profile::
Access your settings, appearance preferences (such as Dark Mode), and logout options via your avatar.

Support & Notifications::
Access help resources and view system alerts.

.Left Hand Navigation Sidebar
The sidebar organizes the platform's core features into distinct sections.

Home::
Returns you to your personalized dashboard.

Catalog::
Browses the inventory of software components, APIs, and resources.

Docs::
Accesses the centralized technical documentation library (TechDocs).

APIs::
Explore and test available APIs.

Create::
Opens the template browser to scaffold new projects.
