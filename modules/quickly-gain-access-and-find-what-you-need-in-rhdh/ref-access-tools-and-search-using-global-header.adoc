= Access tools and search using the Global Header

Use the global header to perform common tasks in {product-very-short}. The header provides access to search, creation tools, and user settings, allowing you to create resources and manage preferences without navigating away from your work.

The header contains the following tools:

Search Input Field::
A universal search bar that instantly queries components, APIs, documentation, and users across the entire platform.

Self-Service (+)::
A quick-action button that opens the template browser, allowing you to bootstrap new projects or register existing components.

Support::
A direct link to your organization's help resources, ticketing system, or documentation.

Notifications::
An alert center that displays updates from plugins, external services, or workflow status changes.

User Profile::
A dropdown menu accessed via your avatar to manage settings, view version information, or log out.

Quick Links::
A set of links to direct access to external applications, such as CI/CD tools, monitoring dashboards, or project management platforms.
