= Finding API definitions in {product-very-short}

To understand endpoints, schemas, and server URLs without accessing source code, locate and inspect API contracts in {product-very-short}. You can review available operations, required parameters, and data models in a readable format.

.Procedure

. In the left navigation sidebar, click *APIs*.
** Alternatively, click *Catalog* and select *API* from the *Kind* filter.

. Select an API from the list to open the detail view.
. Click the *Overview* tab to review metadata, the owner, and lifecycle status.
. Click the *Definition* tab to view the rendered specification.
