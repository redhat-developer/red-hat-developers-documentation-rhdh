= Configure annotations for {aks-short} monitoring by using the Helm chart

[role="_abstract"]
Configure the pod annotations for monitoring {product-short} specific metrics on {aks-name} ({aks-short}) by using the Helm chart.

.Procedure

* Update your `values.yaml` file to annotate the backstage pod for monitoring:
+
[source,yaml]
----
upstream:
  backstage:
    # --- TRUNCATED ---
    podAnnotations:
      prometheus.io/scrape: 'true'
      prometheus.io/path: '/metrics'
      prometheus.io/port: '9464'
      prometheus.io/scheme: 'http'
----

.Verification

* Navigate to the corresponding {azure-short} Monitor Workspace and view the metrics under *Monitoring -> Metrics*.
