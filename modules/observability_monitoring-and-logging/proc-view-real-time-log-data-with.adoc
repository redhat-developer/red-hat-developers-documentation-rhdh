= View real-time log data with {aks-name} ({aks-short})

[role="_abstract"]
View real-time log data from the Container Engine for your {product-short} instance on {aks-short}.

.Prerequisites

* You have deployed {product-short} on {aks-short}.

For more information, see {installing-on-aks-book-link}#assembly-install-rhdh-aks[{installing-on-aks-book-title}].

.Procedure

. Navigate to the {azure-short} Portal.
. Search for the resource group `<your_resource_group>` and locate your {aks-short} cluster `<your_cluster>`.
. Select *Monitoring* -> *Insights* from the menu.
. Go to the *Containers* tab.
. Find the backend-backstage container and click it to view real-time log data as it is generated by the Container Engine.
