= Enable {azure-short} Monitor metrics

[role="_abstract"]
Enable managed Prometheus monitoring for your {aks-name} ({aks-short}) cluster to collect metrics and monitor {product-short} performance through {azure-short} Monitor.

.Prerequisites

* You have an {aks-short} cluster.
* You have the {azure-short} CLI installed.

.Procedure

* To enable managed Prometheus monitoring, use the `--enable-azure-monitor-metrics` option with either the `az aks create` or `az aks update` command:
+
[source,bash]
----
$ az aks create/update --resource-group <your_resource_group> --name <your_cluster> --enable-azure-monitor-metrics
----
+
This command installs the metrics add-on, which gathers https://learn.microsoft.com/en-us/azure/azure-monitor/metrics/prometheus-metrics-overview[Prometheus metrics].

.Verification

* In the {azure-short} portal, navigate to *Monitoring -> Insights* to view the monitoring results. For more information, see https://learn.microsoft.com/en-us/azure/azure-monitor/platform/monitor-azure-resource[Monitor {azure-short} resources with {azure-short} Monitor].
