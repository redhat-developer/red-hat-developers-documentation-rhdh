= RBAC permissions

[role="_abstract"]
Reference information about RBAC permissions for reading, creating, updating, and deleting permission policies and roles, and for accessing homepage features.

[cols="15%,25%,15%,45%", frame="all", options="header"]
|===
|Name
|Resource type
|Policy
|Description

|`homepage.defaultWidgets.read`
|`homepage`
|`read`
|Enables a user or role to view default homepage cards configured by platform engineers

|`homepage.layout.read`
|`homepage`
|`read`
|Enables a user or role to read homepage layout configuration

|`homepage.layout.update`
|`homepage`
|`update`
|Enables a user or role to update their personal homepage layout (if user customization is enabled)

|`policy.entity.read`
|`policy-entity`
|`read`
|Enables a user or role to read permission policies and roles

|`policy.entity.create`
|
|`create`
|Enables a user or role to create a single or multiple permission policies and roles

|`policy.entity.update`
|`policy-entity`
|`update`
|Enables a user or role to update a single or multiple permission policies and roles

|`policy.entity.delete`
|`policy-entity`
|`delete`
|Enables a user or role to delete a single or multiple permission policies and roles
|===
