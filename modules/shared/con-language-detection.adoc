= Language detection

[role="_abstract"]
{product-very-short} detects your language settings across multiple locations to select your {product-very-short} default language.

The system determines the default language by using the following priority order:

. *Browser language priority*: The system first checks your browser language preferences.
. *Configuration priority*: If no browser language matches the supported locales, the `defaultLocale` from the `i18n` configuration in {my-app-config-file} is used as a fallback.
. *Fallback priority*: If neither browser preferences nor configuration provide a match, defaults to `en`.

[NOTE]
====
Once you pick a language, your selection is saved in the user settings.
Later changes to the `{my-app-config-file}` file do not affect the selected language, because you are already using the system.
====
