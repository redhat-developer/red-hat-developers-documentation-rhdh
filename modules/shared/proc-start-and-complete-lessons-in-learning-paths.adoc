= Start and complete lessons in Learning Paths

[role="_abstract"]
As a developer, you can start a course and complete the lessons at your own pace.

.Prerequisites
* You can log in to developers.redhat.com
* Your platform engineer has granted you access to the Learning Paths plugin.


.Procedure

. In your {product} navigation menu, click *Learning Paths*.
. Select the tile for the course you want to begin.
+
[NOTE]
====
This action redirects you to the main page of the course in link:https://developers.redhat.com[the {rhdeveloper-name}s site].
====
