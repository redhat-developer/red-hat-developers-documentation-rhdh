= Review the YAML configuration of your {product} Software Catalog

[role="_abstract"]
You can view the Software Catalog YAML file in your {product} instance to review the metadata for the components in your Software Catalog.

.Procedure

. In your {product} navigation menu, click *Catalog*.
. Find the software component that you want to view, under *Actions*, click the *View* icon.

+
[NOTE]
====
These steps redirect you to the YAML file on your remote repository.
====
