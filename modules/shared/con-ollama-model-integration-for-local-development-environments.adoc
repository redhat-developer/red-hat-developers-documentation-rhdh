= Ollama model integration for local development environments

[role="_abstract"]
When running {product-short} locally for development or prototyping, you can use the open-source Ollama framework to host large language models (LLMs) on your workstation for local testing with {ls-short}. 

This configuration is intended strictly for local evaluation and must not be used for cluster-deployed production environments. 

The Ollama server operates as a containerized layer, providing a command-line interface (CLI) to download, manage, and execute open-source models such as Llama 3 and Mistral. Deploying Ollama as a local container provides a self-contained environment to validate model configurations before transitioning to enterprise inference servers.
