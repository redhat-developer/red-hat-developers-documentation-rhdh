= Prepare your application to add custom navigation and toolbar items

[role="_abstract"]
Install and integrate the global header plugin into your {product} application as a prerequisite for adding custom buttons, menus, and navigation shortcuts to the header.

.Procedure

. To enable custom navigation, install the global header plugin package.
+
[source,bash]
----
yarn --cwd packages/app add @red-hat-developer-hub/backstage-plugin-global-header
----

. To register the global header, import the plugin and module in your `packages/app/src/App.tsx` file.
+
[source,typescript]
----
import { createApp } from '@backstage/frontend-defaults';
import globalHeaderPlugin, {
  globalHeaderModule,
} from '@red-hat-developer-hub/backstage-plugin-global-header/alpha';

export default createApp({
  features: [
    globalHeaderModule,
    globalHeaderPlugin,
  ],
});
----

. Restart your development server to load the plugin.
