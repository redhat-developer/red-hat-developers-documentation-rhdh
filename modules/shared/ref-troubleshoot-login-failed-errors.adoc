= Troubleshoot login failed errors

[role="_abstract"]
When a user cannot sign in to {product-short}, the sign-in page displays a "Login failed" error message.

Common causes of login failures include:

* User identity cannot be resolved in the software catalog
* Authentication provider is not configured for sign-in
* User profile does not contain required attributes such as email
* Catalog provider plugins are not syncing users correctly

For detailed diagnostic steps and solutions for specific login error messages, see xref:diagnose-specific-login-failures_{context}[Diagnose specific login failures].
