= View installed plugins

[role="_abstract"]
Using the Dynamic Plugins Info front-end plugin, you can view plugins that your {product} application currently has installed. {product} enables the Dynamic Plugins Info plugin by default.

.Procedure

. Open your {product-short} application and click *Administration* > *Extensions*.
. Go to the *Installed* tab to view a list of installed plugins and related information.
