= Install the Operator on {platform-long} by using OLM

[role="_abstract"]
Install the {product} Operator on {platform} by using the Operator Lifecycle Manager (OLM) framework.

include::snip-installing-the-operator-common-introduction.adoc[]

.Prerequisites
include::snip-installing-the-operator-common-prerequisites.adoc[]
* You have logged in to your Google account and created a https://cloud.google.com/kubernetes-engine/docs/how-to/creating-an-autopilot-cluster[{gke-short} Autopilot] or https://cloud.google.com/kubernetes-engine/docs/how-to/creating-a-zonal-cluster[{gke-short} Standard] cluster.

.Procedure
* Connect to your {gke-short} cluster:
+
[source,terminal]
----
$ gcloud container clusters get-credentials <cluster_name> --location=<cluster_location>
----
_<cluster_name>_:: Enter your {gke-short} cluster name.
_<cluster_location>_:: Enter your {gke-short} cluster location.
