= Learn platform features through guided tutorials

[role="_abstract"]
Add interactive quick start tutorials to {product} that guide you through key features with step-by-step walkthroughs automatically displayed on first visit and accessible from a sidebar button.

The quick start plugin provides role-based tutorials that help you discover software templates, explore the catalog, and learn platform workflows without reading documentation. The plugin integrates with the app drawer, automatically opens on first visit, and filters tutorials based on your role.

.Prerequisites

* You have installed the `@red-hat-developer-hub/backstage-plugin-quickstart` package.
* You have access to the `packages/app/src/App.tsx` file.

.Procedure

. To enable guided tutorials, import the quick start modules in your `packages/app/src/App.tsx` file:
+
[source,typescript]
----
import { createApp } from '@backstage/frontend-defaults';
import {
  quickstartInitModule,
  quickstartTranslationsModule,
} from '@red-hat-developer-hub/backstage-plugin-quickstart/alpha';

export default createApp({
  features: [
    quickstartInitModule,  # Handles first-visit auto-open and notifications
    quickstartTranslationsModule,  # Provides internationalization support
    // Additional features
  ],
});
----

. To make tutorials accessible, add the quick start sidebar button to your navigation component:
+
[source,typescript]
----
import { useAppDrawer } from '@backstage/frontend-plugin-api';
import { WavingHandOutlinedIcon } from '@mui/icons-material';
import { QUICKSTART_DRAWER_ID } from '@red-hat-developer-hub/backstage-plugin-quickstart';

function QuickstartSidebarItem() {
  const { toggleDrawer } = useAppDrawer();
  return (
    <SidebarItem
      text="Quick start"
      icon={WavingHandOutlinedIcon}
      onClick={() => toggleDrawer(QUICKSTART_DRAWER_ID)}
    />
  );
}
----

.Verification

. Open your {product} instance in a web browser for the first time.
. Verify that the quick start drawer opens automatically on first visit.
. Click the quick start sidebar button to toggle the drawer.
. Confirm that tutorials display based on your role.
