= Install plugins by using Extensions

[role="_abstract"]
You can install and configure plugins by using Extensions.

.Prerequisites
* You have configured {product-very-short} to allow plugins installation from *Extensions*.
* You have {configuring-book-link}[added a custom {product-short} application configuration].

.Procedure
. Navigate to *Extensions*.
. Select a plugin to install.
. Click the *Install* button.
+
image::extend_installing-and-viewing-plugins-in-rhdh/extensions-install-plugin-1.png["Installing a plugin from the Extensions page"]
The code editor is displayed that displays the plugin default configuration.
. Update the plugin configuration, if necessary.
+
image::extend_installing-and-viewing-plugins-in-rhdh/extensions-install-plugin-2.png["Plugin configuration code editor"]
. Click *Install*
. To view the plugins that require a restart, click *View plugins* in the alert message.
+
image::extend_installing-and-viewing-plugins-in-rhdh/extensions-install-plugin-3.png["Alert showing plugins that require a restart"]
. Restart your {product-very-short} application.

.Verification
. After you restart your {product-very-short} application, navigate to *Extensions*.
. Select the plugin that you have installed.
. The *Actions* button is displayed.
// . Click on *Edit* to view the configuration that you used to install the plugin.
