[IMPORTANT]
====
When you use the `{{inherit}}` tag, z-stream updates automatically update the plugin to the latest version from the base configuration. If a plugin update causes issues, you can pin the plugin to a specific version. For more information, see {upgrading-book-link}#rollback-a-plugin-to-a-previous-version_upgrade-rhdh[Roll back a dynamic plugin to a previous version].
====
