= Get instant help with an AI-powered virtual assistant

[role="_abstract"]
Add an AI virtual assistant to {product} that answers questions, debugs issues, and guides you through tasks using natural language. The virtual assistant is accessible through a floating action button that opens an app drawer.

.Prerequisites

* You have installed the `@red-hat-developer-hub/backstage-plugin-lightspeed` package.
* You have access to the `packages/app/src/App.tsx` file.

.Procedure

. Import and register the Lightspeed modules in your `packages/app/src/App.tsx` file:
+
[source,typescript]
----
import { createApp } from '@backstage/frontend-defaults';
import {
  lightspeedFABModule,
  lightspeedTranslationsModule,
} from '@red-hat-developer-hub/backstage-plugin-lightspeed/alpha';

export default createApp({
  features: [
    appDrawerModule,  // Required for app drawer integration
    lightspeedFABModule,  // Provides the floating action button
    lightspeedTranslationsModule,  // Adds internationalization support
    signInModule,
    navModule,
    // Additional features
  ],
});
----

. Save the file and restart your {product} instance.

.Verification

. Open your {product} instance in a web browser.
. Verify that the Lightspeed floating action button appears on the page.
. Click the floating action button to open the virtual assistant interface.
. Enter a question to confirm that the virtual assistant responds with AI-generated answers.
