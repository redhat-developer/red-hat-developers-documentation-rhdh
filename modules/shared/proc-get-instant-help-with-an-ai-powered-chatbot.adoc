= Get instant help with an AI-powered chatbot

[role="_abstract"]
Add an AI chatbot to {product} that answers questions, debugs issues, and guides users through tasks using natural language, accessible through a floating action button that integrates with the app drawer.

The Lightspeed plugin provides a conversational AI assistant that helps users troubleshoot problems and learn platform features without leaving their current page. The chatbot appears in an app drawer, controlled by a floating action button that displays contextually—showing a close icon when active and the Lightspeed icon when inactive.

.Prerequisites

* You have installed the `@red-hat-developer-hub/backstage-plugin-lightspeed` package.
* You have access to the `packages/app/src/App.tsx` file.

.Procedure

. Import and register the Lightspeed modules in your `packages/app/src/App.tsx` file:
+
[source,typescript]
----
import { createApp } from '@backstage/frontend-defaults';
import {
  lightspeedFABModule,
  lightspeedTranslationsModule,
} from '@red-hat-developer-hub/backstage-plugin-lightspeed/alpha';

export default createApp({
  features: [
    appDrawerModule,  # Required for app drawer integration
    lightspeedFABModule,  # Provides the floating action button
    lightspeedTranslationsModule,  # Adds internationalization support
    signInModule,
    navModule,
    // Additional features
  ],
});
----

. Save the file and restart your {product} instance.

.Verification

. Open your {product} instance in a web browser.
. Verify that the Lightspeed floating action button appears on the page.
. Click the *Floating action button* to open the chatbot drawer.
. Confirm that the button icon changes from the Lightspeed icon to a close icon when the chatbot is active.
. Ask the chatbot a question to verify it responds with AI-generated answers.
. Click the *Close* icon to close the chatbot drawer.
