= UI elements with RBAC visibility control

[role="_abstract"]
The following table describes the UI elements that support permission-based visibility control.

[cols="20%,30%,50%", frame="all", options="header"]
|===
|UI Element
|Blueprint
|Behavior

|Pages
|`PageBlueprint`
|Controls whether a route is registered and whether the page is accessible. Also controls the visibility of the corresponding navigation item.

|Entity Tabs
|`EntityContentBlueprint`
|Controls the visibility of tabs within the entity information view. Tabs are hidden if the user lacks the required permissions.

|Entity Cards
|`EntityCardBlueprint`
|Controls whether a card is rendered on an entity page. Cards are hidden if the user lacks the required permissions.

|Homepage Widgets
|`HomePageWidgetBlueprint`
|Controls whether a widget is displayed on the homepage. Widgets are hidden if the user lacks the required permissions.

|Navigation Items
|`NavItemBlueprint`
|Controls whether a navigation item appears in the sidebar. Items are hidden if the user lacks the required permissions.
|===
