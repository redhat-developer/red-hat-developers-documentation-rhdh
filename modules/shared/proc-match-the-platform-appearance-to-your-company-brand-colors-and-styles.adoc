= Match the platform appearance to your company brand colors and styles

[role="_abstract"]
Apply custom color schemes, logos, and visual elements to the {product} interface using the new front-end system so the platform reflects your organization's brand identity.

The new front-end system uses `@backstage/frontend-defaults` to provide modern {backstage} theme architecture.

.Prerequisites

* You have installed the `@backstage/frontend-defaults` package.
* You have access to the `packages/app/src/App.tsx` file.
* You have access to the `{my-app-config-file}` file.

.Procedure

. To define your brand colors, create theme constants with specific color values for your visual identity elements in a `theme-constants.ts` file:
+
[source,typescript]
----
type ThemeInfo = {
  name: 'Custom Light' | 'Custom Dark';
  primaryColor: string;
  headerColor1: string;
  headerColor2: string;
  navigationIndicatorColor: string;
};

const customLightTheme: ThemeInfo = {
  name: 'Custom Light',
  primaryColor: 'rgb(0, 102, 204)',
  headerColor1: 'rgb(0, 51, 102)',
  headerColor2: 'rgb(0, 102, 204)',
  navigationIndicatorColor: 'rgb(0, 102, 204)',
};
----

. To apply the theme constants, create a custom theme provider using *Material-UI* theming:
+
[source,typescript]
----
import { createTheme, ThemeProvider } from '@material-ui/core/styles';

const lightTheme = createTheme({
  palette: {
    primary: {
      main: customLightTheme.primaryColor,
    },
  },
});
----

. To enable the custom theme, import and configure theme extensions from the theme plugin alpha module in your `packages/app/src/App.tsx` file:
+
[source,typescript]
----
import { createApp } from '@backstage/frontend-defaults';
import { rhdhThemeModule } from '@red-hat-developer-hub/backstage-plugin-theme/alpha';

export default createApp({
  features: [
    rhdhThemeModule,
    // Additional NFS features
  ],
});
----

. To set the application title and branding, configure extension settings in your `{my-app-config-file}` file:
+
[source,yaml]
----
app:
  title: My Company Catalog
  baseUrl: http://localhost:3000

organization:
  name: My Company
----

.Verification

. Start your application in the new front-end system mode.
. Verify that the interface displays your custom theme.
. Check that theme colors match your brand identity specifications.
