= Navigate the plugin marketplace and filter plugins using badges

[role="_abstract"]
The plugin marketplace uses badges and indicators to display plugin status, support level, and maturity to help you make informed decisions when selecting plugins.

include::{docdir}/artifacts/snip-developer-preview.adoc[]

include::{docdir}/artifacts/snip-technology-preview.adoc[]

The marketplace employs the following badges and tags to offer quick visual cues:

* *Certified*
* *Generally Available* (*GA*)
* *Community Plugin*
* *Tech Preview*
* *Dev Preview*
* *Custom Plugin*
+
[NOTE]
====
Hovering your mouse cursor over any badge or tag (such as *GA* or *Certified*) on either the plugin list or details page displays a helper text explanation clarifying its meaning and implications. Hovering over tags such as *{technology-preview}*, or *{developer-preview}* will display a helper text explanation clarifying its meaning and indicating if it is in a technical preview phase or a development preview phase.
====

The plugin details pages include the following fields for additional context: `author`, `tags`, `category`,`publisher`, and `support provider`. The update also changes the `version` text to `{product-custom-resource-type} compatibility version` to offer more precise compatibility information.

The following filter examples are available:

* To view only plugins that are *Generally Available* (*GA*), apply the *GA* filter.

* To find *Certified* plugins, apply the `Certified` in *Support type* filter.

* To explore plugins currently in *Tech Preview*, *Dev Preview*, or *Community Plugin* status, apply the corresponding filter.

.Procedure

. Navigate to the main plugin list page within the {product}.

. On the plugin list page, use the *Support type* filter to refine your plugin listings based on the plugin support status. You can use this filter to quickly narrow down options to plugins that meet your specific support requirements.

. Click the entry of a plugin list page to view its details, which offer an in-depth view of its characteristics.
