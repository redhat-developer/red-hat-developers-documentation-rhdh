= Disable the Guest login

To prevent users from logging in as a guest on the login page, disable the guest login option.

.Procedure

. In the `{my-app-config-file}` file, set the authentication environment to `production`:
+
[source,yaml]
----
auth:
  environment: production
----

. Restart the {product-short} application to apply the changes.

.Verification

. Go to the login page of your {product-short} instance.

. Verify that the option to log in as a guest is no longer available.
