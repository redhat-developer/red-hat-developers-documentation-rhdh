= Display a complete list of Orchestrator infrastructure values with Helm CLI

[role="_abstract"]
Use the available options to configure Orchestrator infrastructure with Helm Chart: the Helm deployment method specific configuration files.

.Procedure

. Pull the released {product-very-short} Orchestrator Infrastructure Helm Chart, including all its dependencies:
+
[,console,subs="+attributes,+quotes"]
----
helm pull redhat-developer-hub-orchestrator-infra
  --repo https://charts.openshift.io
  --version {product-chart-version}
----

. Display the configurable infrastructure values from the chart:
+
[,console,subs="+attributes,+quotes"]
----
helm show values redhat-developer-hub-orchestrator-infra \
  --repo https://charts.openshift.io \
  --version {product-chart-version}
----
