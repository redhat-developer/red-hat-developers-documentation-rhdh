= Viewing pod logs

.Procedure
* To view pod logs, you must grant the following permission to the `ClusterRole`:
+
[source,yaml]
----
 apiVersion: rbac.authorization.k8s.io/v1
  kind: ClusterRole
  metadata:
    name: backstage-read-only
  rules:
    ...
    - apiGroups:
        - ''
      resources:
        - pods
        - pods/log
      verbs:
        - get
        - list
        - watch
----
