= Front-end plugin wiring

Front-end plugin wiring integrates dynamic front-end plugin components, such as new pages, UI extensions, icons, and APIs, into {product}.

Because the dynamic plug-ins load at runtime, the core application must discover and connect the exported assets of the plug-in to the appropriate user interface systems and locations.
