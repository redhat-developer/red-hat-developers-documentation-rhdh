= Enabling the Nexus Repository Manager plugin

The Nexus Repository Manager plugin is pre-loaded in {product-short} with basic configuration properties. To enable it, set the disabled property to `false` as follows:

[source,yaml,subs="+quotes"]
----
global:
  dynamic:
    includes:
      - dynamic-plugins.default.yaml
    plugins:
      - package: oci://ghcr.io/redhat-developer/rhdh-plugin-export-overlays/backstage-community-plugin-nexus-repository-manager:__<tag>__
        disabled: false
----
The `<tag>` variable is your RHDH application{apos}s version of Backstage and the plugin version, in the format: bs_<backstage-version>_<plugin-version>. To find the versions, see the link:https://github.com/redhat-developer/rhdh-plugin-export-overlays[rhdh-plugin-export-overlays repository].
