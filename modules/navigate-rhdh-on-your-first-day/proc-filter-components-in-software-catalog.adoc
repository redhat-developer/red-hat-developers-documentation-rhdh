= Filter components in the Software Catalog

To identify existing software building blocks, view source code, or check lifecycle status, use the catalog filters in {product-very-short}. Filtering the catalog makes sure you can locate production-ready components and documentation without performing manual searches across multiple repositories.

.Procedure

. Log in to the {product-very-short} interface.
. In the navigation sidebar, click *Catalog*.
. Click the *Kind* filter and select *Component*.
. Optional: Click the *Type* filter to select a functional category, such as *Library*, *Website*, or *Tool*.
. In the *Search* field, enter the name of the component.
. Select the component card to view the *Overview*, *Docs*, and *Relationships* tabs.
