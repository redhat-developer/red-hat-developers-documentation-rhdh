= Enable the source code editor

[role="_abstract"]
Enable the source code editor to allow developers to open source code directly from {product-very-short}.

.Procedure

. Grant read access to the CheClusters resource in the `ClusterRole`:
+
[source,yaml]
----
 ...
  apiVersion: rbac.authorization.k8s.io/v1
  kind: ClusterRole
  metadata:
    name: backstage-read-only
  rules:
    ...
    - apiGroups:
        - org.eclipse.che
      resources:
        - checlusters
      verbs:
        - get
        - list
----

. Add the following configuration to the `kubernetes.customResources` property in your `{my-app-config-file}` file:
+
[source,yaml]
----
 kubernetes:
    ...
    customResources:
      - group: 'org.eclipse.che'
        apiVersion: 'v2'
        plural: 'checlusters'
----
