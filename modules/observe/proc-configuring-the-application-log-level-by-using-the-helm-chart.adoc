= Configuring the log level by using the {product} Helm chart

You can configure the application log level with the {product} Helm chart. This sets the minimum severity level of events for your observability tool to log and analyze. 

.Procedure
* Modify the logging level by adding the environment variable `LOG_LEVEL` to your Helm chart `values.yaml` file. For example:
+
[source,yaml]
----
upstream:
  backstage:
    # --- Truncated ---
    extraEnvVars:
      - name: LOG_LEVEL
        value: debug
----
