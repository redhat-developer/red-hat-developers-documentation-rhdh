= Configuring the application log level by using the {product} Helm chart

Using the {product} Helm chart, you can configure the application log level through the environment variable `LOG_LEVEL`. This sets the minimum severity level of events for your application to log.

.Procedure
* Add `LOG_LEVEL`
+
[source,yaml]
----
upstream:
  backstage:
    # --- Truncated ---
    extraEnvVars:
      - name: LOG_LEVEL
        value: debug
----
