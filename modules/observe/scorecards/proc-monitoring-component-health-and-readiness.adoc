= Monitoring component health and readiness with Scorecard metrics

You can view metrics to evaluate the health and security of your software components directly in the {product-very-short} catalog.

.Prerequisites

* You have installed your {product-very-short} instance.
* You have configured the GitHub or Jira Scorecard (or both) plugin.
* You must have permissions to read the Scorecard metrics.

.Procedure
. In your {product-very-short} navigation menu, go to *Catalog*.
. Select the software component (catalog entity) that has Scorecard metrics configured.
. On the component *Service* page, click the *Scorecard* tab.
. Select a metric tile to view detailed data.
