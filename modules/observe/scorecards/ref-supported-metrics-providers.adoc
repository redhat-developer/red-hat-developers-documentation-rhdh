= Supported Scorecard metrics providers

The Scorecard plugin collects data from third-party sources by using metric providers. Use the following table to identify the metrics available for each supported provider.

The following metric providers are supported:

[cols="1,1,2,3,1", options="header"]
|===
| Provider | Metric ID | Title | Description | Type

| GitHub
| `github.open_prs`
| GitHub open PRs
| Number of open pull requests in GitHub.
| number

| Jira
| `jira.open_issues`
| Jira open issues
| Number of open issues in Jira.
| number
|===
