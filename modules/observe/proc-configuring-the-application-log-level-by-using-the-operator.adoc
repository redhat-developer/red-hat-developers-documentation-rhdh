= Configuring the log level by using the {product} Operator

You can configure the application log level with the {product} Operator. This sets the minimum severity level of events for your observability tool to log and analyze.

.Procedure
* Modify the logging level by including the environment variable `LOG_LEVEL` in your custom resource as follows:
+
[source,yaml]
----
spec:
  # Other fields omitted
  application:
    extraEnvs:
      envs:
        - name: LOG_LEVEL
          value: debug
----
