= Providing custom TechDocs add-ons

[role="_abstract"]
If a plugin provides many add-ons, each `techdocsAddon` entry specifies a unique `importName` corresponding to the add-on.
Front-end plugins expose the TechDocs add-on component by using the `techdocsAddons` configuration as shown in the following example:

[source,yaml]
----
dynamicPlugins:
  frontend:
    my-plugin: # The plugin package name
      techdocsAddons:
        - importName: ExampleAddon # The exported add-on component
          config:
            props: ... # (Optional): React props to pass to the add-on
----
