== Retrieving software template metadata

The `software-template-metadata-tool` retrieves metadata for a specific software template.

.software-template-metadata-tool parameters
[cols="30%,70%",options="header"]
|===
|Parameter |Description
|`templateRef` |The reference identifier for the software template.
|===
