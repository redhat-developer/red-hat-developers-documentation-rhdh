= Resolving authentication issues when tools are not found

If you encounter a `500 Server Error` when unregistering catalog entities, perform the following steps:

.Procedure

. Verify that the catalog location URL or the UUID is valid.
. Verify that your authentication token has sufficient permissions to modify the catalog.
. Check the `software-catalog-mcp-tool` logs for specific error details.
