== Unregistering entities using catalog-unregister-tool

Use the `catalog-unregister-tool` to remove an existing entity from the Software Catalog.

.Parameters for catalog-unregister-tool
[cols="30%,70%",options="header"]
|===
|Parameter |Description
|`location` |A valid catalog location URL or a UUID.
|===
