== Unregistering entities using catalog-unregister-tool

The `catalog-unregister-tool` removes an existing entity from the software catalog.

.catalog-unregister-tool parameters
[cols="30%,70%",options="header"]
|===
|Parameter |Description
|`location` |A valid catalog location URL or a UUID.
|===
