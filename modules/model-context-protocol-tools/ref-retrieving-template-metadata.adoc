= Retrieving Software Template metadata

Use the `software-template-metadata-tool` to retrieve metadata for a specific Software Template.

.Parameters for `software-template-metadata-tool`
[cols="30%,70%",options="header"]
|===
|Parameter |Description
|`templateRef` |The reference identifier for the Software Template.
|===
