= View the Adoption Insights card

[role="_abstract"]
View detailed analytics through individual dashboard cards including active users, total users, and top catalog entities.

[NOTE]
====
For the following Adoption Insights Top List widgets, the *Name* column displays the `metadata.title` when available:

* Top catalog entities

* Top templates

* Top plugins

* Top TechDocs

If the `metadata.title` is missing or empty, the UI falls back to:

* `metadata.name` for entities, or

* `plugin_id` for plugins.

Hover over the entity or plugin name tooltip to display the raw identifier (`metadata.name` or `plugin_id`).
====

.Procedure

* Navigate to *Administration -> Adoption Insights* to access the dashboard cards.
