Table 3 lists recommended PostgreSQL performance tuning parameters based on the deployment scale for an external PostgreSQL deployment.
Set `shared_buffers` to approximately 1/4 of the allocated database memory and `effective_cache_size` to approximately 1/2 of the allocated database memory.
For more information, see the link:https://wiki.postgresql.org/wiki/Tuning_Your_PostgreSQL_Server[PostgreSQL tuning guide].
