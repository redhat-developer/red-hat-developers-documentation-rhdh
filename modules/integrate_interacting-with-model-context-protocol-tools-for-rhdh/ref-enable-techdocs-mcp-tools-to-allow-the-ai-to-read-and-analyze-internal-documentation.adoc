= Enable TechDocs MCP tools to allow the AI to read and analyze internal documentation

[role="_abstract"]
Enable and use TechDocs MCP tools to allow the AI assistant to search, read, and analyze your internal technical documentation to resolve queries with accurate portal data.

The TechDocs MCP tool enables MCP clients to search and retrieve documentation from {product-very-short} for use as context in AI applications.

The following table lists the TechDocs tools and parameters:

[cols="25%,45%,30%",options="header"]
|===
|Tool |Description |Parameters
|`fetch-techdocs` |Lists all entities with registered TechDocs. Includes metadata such as `lastModified` and build information. |`entityType`, `namespace`, `owner`, `lifecycle`, `tags`
|`analyze-techdocs-coverage` |Calculates the percentage of entities with TechDocs configured to identify documentation gaps. |`entityType`, `namespace`, `owner`, `lifecycle`, `tags`
|`retrieve-techdocs-content` |Retrieves the content of a specific TechDocs resource. |`entityRef`, `pagePath`
|===
