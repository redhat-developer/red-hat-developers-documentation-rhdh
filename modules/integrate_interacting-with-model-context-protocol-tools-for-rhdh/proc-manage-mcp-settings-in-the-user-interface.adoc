= Manage MCP settings in the user interface

[role="_abstract"]
Manage Model Context Protocol (MCP) server connections and personal authentication tokens directly from the {ls-brand-name} chat interface.

You can use the *MCP settings* panel to review server status and enable or disable specific tools for your session.

.Procedure

. In the {ls-brand-name} chat box, click the menu icon in the header.
. Select *MCP settings*.
. In the *MCP settings* panel, perform any of the following actions:
** Review server status: View the connectivity state of available MCP servers.
** Configure personal tokens: 
*** To add or update a token, click the *Edit* icon, enter your Personal Access Token in the field, and click *Save*. The system validates the token automatically.
*** To remove or replace a token, click the *Edit* icon and select *Forget Token*.
** Enable or disable servers: Use the toggles to choose which servers provide tools for your chat session.
+
[NOTE]
====
You must configure a token before you can enable a server.
====

.Verification

* Confirm the status message displayed below the token field:
** *Connection successful*: The token is valid and verified.
** *Authorization failed. Try again*: The token is incorrect, improperly formatted, or the connection failed. You must verify the token and your network connection.
