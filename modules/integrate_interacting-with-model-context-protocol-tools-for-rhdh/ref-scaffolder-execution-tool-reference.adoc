= Scaffolder execution tool reference

[role="_abstract"]
Use this tool to trigger the execution of a Software Template.

The following table describes the inputs fields:

[cols="1,2,1", options="header"]
|===
| Input parameter | Description | Example
| `templateRef` | Entity reference of the target template. | `template:default/create-node`
| `values` | Key-value map of required input values. | `{"owner": "team-a"}`
| `secrets` | Optional: Secrets required for execution. | `{"githubToken": "ghp_abc"}`
|===

The following table describes the output fields:

[cols="1,3"]
|===
| Output field | Description
| `taskId` | ID of the created task used for log tracking.
|===
