= Install MCP server plugins

[role="_abstract"]
To run MCP tools within {product-very-short}, you must add the MCP server plugin to your configuration. This installation enables the backend infrastructure required to manage MCP actions.

.Prerequisites

* Your {product-very-short} instance is installed and running.

.Procedure

* In your dynamic plugins ConfigMap (for example, `dynamic-plugins-rhdh.yaml`), add the updated MCP server plugin:
+
[source,yaml]
----
plugins:
    - package: oci://ghcr.io/redhat-developer/rhdh-plugin-export-overlays/backstage-plugin-mcp-actions-backend:<tag>
      disabled: false
----
+
include::{docdir}/artifacts/snip-tag-for-OCI-package-paths.adoc[]

.Verification

* Confirm the plugins are active by checking the {product-very-short} logs for "loaded" status messages, or by verifying that the corresponding MCP tools appear in your product tool registry.
