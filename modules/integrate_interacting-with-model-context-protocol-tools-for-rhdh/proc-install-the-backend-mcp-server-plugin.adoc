= Install the backend MCP server plugin

[role="_abstract"]
The backend MCP server plugin is responsible for running MCP tools within {product-very-short}.

.Procedure

* In your dynamic plugins ConfigMap (for example, `dynamic-plugins-rhdh.yaml`), add the updated MCP server plugin:
+
[source,yaml]
----
plugins:
    - package: oci://ghcr.io/redhat-developer/rhdh-plugin-export-overlays/backstage-plugin-mcp-actions-backend:bs_1.49.4__0.1.11!backstage-plugin-mcp-actions-backend
      disabled: false
----
