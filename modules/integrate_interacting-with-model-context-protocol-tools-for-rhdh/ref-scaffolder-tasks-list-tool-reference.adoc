= Scaffolder tasks list tool reference

[role="_abstract"]
Use the Scaffolder tasks list tool to retrieve and filter a list of Scaffolder tasks.

The following table describes the input fields:

[cols="1,2,1", options="header"]
|===
| Input parameter | Description | Example
| `owned` | If `true`, returns only tasks created by the current user. | `true`
| `limit` | Maximum number of tasks to return per request. | `5`
| `offset` | Number of tasks to skip for pagination. | `10`
|===

The following table describes the output fields:

[cols="1,2,1", options="header"]
|===
| Output field | Description | Example
| `tasks[]` | A list of Scaffolder tasks including ID, timestamps, and status. Statuses include: `open`, `processing`, `completed`, `failed`, `cancelled`, and `stale`. | `{"id": "b8e7f2a1...", "status": "processing"}`
|===
