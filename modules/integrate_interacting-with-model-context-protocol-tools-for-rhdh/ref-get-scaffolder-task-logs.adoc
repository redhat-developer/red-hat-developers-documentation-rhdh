= Get Scaffolder task logs

[role="_abstract"]
Use this tool to retrieve log events for a task to monitor progress or troubleshoot execution.

The following table describes the input parameters:

[cols="1,2,1", options="header"]
|===
| Input parameter | Description | Example
| `taskId` | Unique identifier for a task. | `b8e7f2a1-3c4d-4e6f-8a9b-0c1d2e3f4a5b`
| `after` | *Optional:* Return only events after this event ID. | `1847`
|===

The following table describes the output parameters:

[cols="1,2,1", options="header"]
|===
| Output parameter | Description | Example
| `log` | The log output for the specified task ID. Each log entry includes the message, `stepId`, and step status. Event types include `log`, `completion`, `cancelled`, or `recovered`. | `"2026-03-27... Finished step Fetch skeleton"`
|===
