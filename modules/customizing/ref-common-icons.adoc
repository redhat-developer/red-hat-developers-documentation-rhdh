= Common icons for customization

Use the following icons to customize the {product} user interface (UI), including the global header, quickstarts, and quick-access cards. 

To use these icons in configuration files, you must use the exact string provided in the following list.

[IMPORTANT]
====
Icon keys are case-sensitive. You must enter them exactly as shown in the following list to ensure they render correctly in the UI.
