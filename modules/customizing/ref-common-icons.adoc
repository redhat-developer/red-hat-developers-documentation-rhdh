= Common icons for customization

The following icons are available for use when customizing the {product} user interface, including the global header, quickstarts, and quick-access cards. 

To use these icons in your configuration files, you must use the exact string provided in the *Icon Key* column.

.Available common icons
[cols="1,2", options="header"]
|===
| Icon Key | Description
| `GitHub` | The GitHub logo.
| `GitLab` | The GitLab logo.
| `Bitbucket` | The Bitbucket logo.
| `Azure` | The Microsoft Azure logo.
| `Jenkins` | The Jenkins automation server logo.
| `Slack` | The Slack communication platform logo.
| `Discord` | The Discord communication platform logo.
| `Google` | The Google logo.
| `Email` | A generic envelope icon for email.
| `Link` | A generic chain-link icon for external URLs.
| `Help` | A question mark icon for documentation or support.
|===

[IMPORTANT]
====
Icon keys are case-sensitive. You must enter them exactly as shown in the table to ensure they render correctly in the UI.
====
