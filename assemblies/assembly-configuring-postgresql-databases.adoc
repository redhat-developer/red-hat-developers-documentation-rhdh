= Configuring external PostgreSQL databases

As an administrator, you can configure and use external PostgreSQL databases in {product}. You must have a PostgreSQL certificate file that is used to configure the {product-very-short} instance either using the Operator or Helm Chart.

[NOTE]
====
{product-very-short} supports only configuring PostgreSQL databases. You are responsible for maintenance activities, such as backing up your data or configuring high availability (HA) for the configured PostgreSQL databases.
====
