= Configuring external PostgreSQL databases

As an administrator, you can configure and use external PostgreSQL databases in {product}. Based on your needs, you can use a PostgreSQL certificate file to configure an external PostgreSQL instance using the Operator or Helm Chart.

[NOTE]
====
{product-very-short} supports only configuring PostgreSQL databases. You are responsible for maintenance activities, such as backing up your data or configuring high availability (HA) for the configured PostgreSQL databases.

Also, configuring PostgreSQL databases by using the {product} Operator is not intended for production use.
====
