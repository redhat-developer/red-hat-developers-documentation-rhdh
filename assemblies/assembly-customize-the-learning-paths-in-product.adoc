= Customize the Learning Paths in {product}

[role="_abstract"]
In {product}, you can configure Learning Paths by hosting the required data externally and using the built-in proxy to deliver this data. You can provide Learning Paths data from a JSON file hosted on a web server or from a dedicated service that provides the data in JSON format using an API.
