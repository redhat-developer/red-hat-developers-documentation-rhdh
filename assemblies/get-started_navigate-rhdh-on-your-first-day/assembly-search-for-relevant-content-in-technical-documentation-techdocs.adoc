= Search for relevant content in Technical Documentation (TechDocs)

:context: search-for-relevant-content-in-technical-documentation-techdocs

[role="_abstract"]
To reduce context switching and to ensure technical resources are accessible, use TechDocs in {product}. By centralizing documentation, you can review architecture diagrams, installation guides, and component details alongside software entities in a single interface.

== Prerequisites

* Your administrator has enabled and configured the TechDocs plugin.
* Documentation is imported into TechDocs.
* You have the required roles and permissions to access TechDocs content.
