= Logging with Amazon CloudWatch logs

Logging within the {product} relies on the link:https://github.com/winstonjs/winston[Winston library]. By default, logs at the debug level are not recorded. To activate debug logs, you must set the environment variable `LOG_LEVEL` to debug in your {product} instance.
