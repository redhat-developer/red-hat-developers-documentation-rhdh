= Monitoring with Amazon Prometheus

{product} provides Prometheus metrics related to the running application. 


.Prerequisites
* You configured Prometheus for {eks-name} ({eks-short}) clusters, see link:https://docs.aws.amazon.com/eks/latest/userguide/prometheus.html[Prometheus metrics] in the Amazon documentation.
* You link:https://docs.aws.amazon.com/prometheus/latest/userguide/AMP-onboard-create-workspace.html[created an Amazon managed service for the Prometheus workspace].
* You link:https://docs.aws.amazon.com/prometheus/latest/userguide/AMP-onboard-ingest-metrics.html[configured Prometheus to ingest of the Developer Hub Prometheus metrics].
* You ingested Prometheus metrics into the created workspace.
