= Extensions in {product}
:context: extensions-in-rhdh

[role="_abstract"]
{product} ({product-very-short}) includes the Extensions feature, which comes preinstalled and enabled by default. The Extensions feature provides users with a centralized interface to browse and manage available plugins

You can use Extensions to discover plugins that extend {product-very-short} functionality, streamline development workflows, and improve the developer experience.
