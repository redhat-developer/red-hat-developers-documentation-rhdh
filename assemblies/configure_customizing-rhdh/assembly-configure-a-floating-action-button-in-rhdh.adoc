= Configure a floating action button in {product}

:previouscontext: {context}
:context: configure-a-floating-action-button-in-rhdh

[role="_abstract"]
Configure any action as a floating button in the {product-short} instance by using the floating action button plugin.

[IMPORTANT]
====
As of {product-very-short} 1.9, the Global Floating Action Button plugin is disabled by default and is deprecated. The Global Header plugin provides equivalent and improved navigation capabilities by using quick links and starred items.

For new deployments, use the Global Header instead. See xref:configure-the-global-header-in-rhdh_customizing-rhdh[Configure the Global Header in {product}]. 

Existing deployments that explicitly enable the Global Floating Action Button are not affected and can continue using this plugin.  
====
