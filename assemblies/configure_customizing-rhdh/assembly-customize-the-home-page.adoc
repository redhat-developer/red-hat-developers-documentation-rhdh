= Customize the Home page

:previouscontext: {context}
:context: customize-the-home-page

[role="_abstract"]
You can customize the {product} Home page by modifying your {product-very-short} `{my-app-config-file}` file to extend the default layout with additional cards, change quick access links, and add, reorganize, or remove available homepage cards.

Platform engineers can use RBAC group-based visibility rules to show different homepage content to different user roles.
