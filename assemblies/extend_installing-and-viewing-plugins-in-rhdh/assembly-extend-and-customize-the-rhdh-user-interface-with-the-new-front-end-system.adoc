= Extend and customize the {product} user interface with the new front-end system

:previouscontext: {context}
:context: extend-and-customize-the-rhdh-user-interface-with-the-new-front-end-system

[role="_abstract"]
Upgrade your {product} environment to the new front-end system to improve extensibility and performance. This migration helps you to customize themes and navigation.

The new front-end system replaces the older `Scalprum` and Webpack Module Federation approach. By migrating your UI features to the new front-end system, you can configure the dynamic homepage, global header, and integrate core plugins directly through your application features.
