= Package and publish {plugin-type-name} plugins as dynamic plugins
:context: package-and-publish-plugins-as-dynamic-plugins

[role="_abstract"]
After exporting a {plugin-type-name} plugin, you can package the derived package into one of the following supported formats:

* Open Container Initiative (OCI) image (recommended)
* TGZ file
* JavaScript package
+
[IMPORTANT]
====
You must publish exported dynamic plugin packages only to private NPM registries.
====
