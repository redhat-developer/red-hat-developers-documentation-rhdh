= Schema-based database isolation implementation

:context: schema-based-database-isolation-implementation

[role="_abstract"]
{product-short} creates a separate database for each plugin. You can use `pluginDivisionMode: schema` to consolidate all plugin data into a single PostgreSQL database, using schemas for isolation instead.
