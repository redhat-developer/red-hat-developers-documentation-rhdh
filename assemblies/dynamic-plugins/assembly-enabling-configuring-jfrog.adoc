= Enabling and configuring the JFrog plugin
:context: assembly-installing-configuring-jfrog

[role="_abstract"]
The JFrog Artifactory plugin is a front-end extension that displays information about container images stored in your JFrog Artifactory repository. As a community-contributed tool, it is listed in the {dynamic-plugins-reference-book-link}#community-plugins-migration-table[Community plugins migration table]. To use the plugin, you must first enable and configure it.
