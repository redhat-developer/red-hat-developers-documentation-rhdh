= Enabling and configuring the JFrog plugin
:context: assembly-installing-configuring-jfrog

JFrog Artifactory is a front-end plugin that displays the information about your container images stored in the JFrog Artifactory repository. The JFrog Artifactory plugin is a community plugin and can be found in the {dynamic-plugins-reference-book-link}#community-plugins-migration-table[Community plugins migration table] in the Dynamic Plugins Reference guide. To use it, you need to enable and configure it.
