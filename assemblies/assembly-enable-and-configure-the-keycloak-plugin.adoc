= Enable and configure the Keycloak plugin
:context: enable-and-configure-the-keycloak-plugin

[role="_abstract"]
Integrate Keycloak into {product} to synchronize users and groups from your {rhbk-brand-name} realm.

The Keycloak backend plugin, which integrates Keycloak into {product-short}, has the following capabilities:

* Synchronization of Keycloak users in a realm.
* Synchronization of Keycloak groups and their users in a realm.

[NOTE]
====
The supported {rhbk-brand-name} ({rhbk}) version is `{keycloak-version}`.
====
