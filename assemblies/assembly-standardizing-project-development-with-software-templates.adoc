= Standardizing project development with software templates

:context: standardizing-project-development

As a platform engineer, you can use software templates in {product-name} to provide standardized project starter kits for your development teams. This self-service model improves developer productivity and ensures that new projects follow organizational standards and best practices.

.Prerequisites

* You have administrator access to a {product-name} instance.
* You have a Git repository to store template files.
