= Automate Software Templates by using Scaffolder MCP tools
:context: automate-software-templates-by-using-scaffolder-mcp-tools

[role="_abstract"]
Use the Scaffolder Model Context Protocol (MCP) (`scaffolder-mcp-extras`) tools to facilitate Software Template development and troubleshooting in {product-very-short}.

You can configure these tools to enable developers to query actions, dry-run templates, and fetch tasks. These tools use provided credentials and adhere to existing role-based access control (RBAC) permissions.

[IMPORTANT]
====
All operations performed through Scaffolder MCP tools use the `On Behalf Of the User` (OBOU) security model. The system uses the bearer token from the MCP client to ensure that actions adhere to existing RBAC permissions and maintain a consistent audit trail.
====
